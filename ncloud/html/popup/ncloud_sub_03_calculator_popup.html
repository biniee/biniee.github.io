Server

  • 운영체제
  • 서버 스펙
  • 추가 스토리지
  • 서버 대수
  • Local Disk 타입의 서버는 서버 요금에 스토리지 요금(운영체제 및 데이터 영역)이 함께 청구됩니다.
  • 추가 스토리지는 10GB 당 한 달에 1,152원(30일 기준)이 청구됩니다.
  • 서버 스펙 및 상세 요금은 Server 상품 소개를 참고하시기 바랍니다.

Network

네트워크 트래픽을 서버의 성능과 부하량을 고려하여 다수의 서버로 분산시켜주는 서비스입니다

총 요금(1개월) 86,400원 추가하기
  • 공인 IP0 원
  • Load Balancer0
  • 네트워크(국내 회선) 사용량4,321,600
  • 네트워크(해외 회선) 사용량4,321,600
  • 공인 IP는 개당 한 달에 5,040원(30일 기준)이 청구됩니다.
  • Load Balancer는 개당 한 달에 21,600원(30일 기준)이 청구됩니다.
  • 네트워크는 아웃바운드 트래픽만 과금되며, 사용량에 따라 차등을 두어 청구됩니다.
  • 네트워크 상품에 대한 더 자세한 내용은 Network 상품 소개를 참고하시기 바랍니다.

  • 사용량에 따라 차등 청구됩니다.
  • 파일 스토리지 상품에 대한 더 자세한 내용은 File Storage 상품 소개를 참고하시기 바랍니다.

총 요금(1개월) 86,400원 추가하기
  • 사용량0
  • 요청 수21,600
  • CDN 요금은 전송량과 요청 수를 합산해서 청구됩니다.
  • 사용량에 따라 차등 청구됩니다.
  • 요청 수는 10,000건당 10원의 요금이 적용됩니다.
  • CDN 상품에 대한 더 자세한 내용은 CDN 상품 소개를 참고하시기 바랍니다.

NAS

다수의 서버가 대용량의 스토리지를 네트워크로 연결해 사용할 수 있는 스토리지 서비스입니다

총 요금(1개월) 888,888,888원 추가하기
  • 사용량31,600
  • 최소 생성 용량은 500GB이며, 100GB씩 추가할 수 있습니다.
  • 기본 용량 500GB를 한 달(30일 기준) 동안 이용하면 월간 이용료로 약 57,600원이 발생합니다.
  • 추가 용량 100GB를 한 달(30일 기준) 동안 이용하면 월간 이용료로 약 11,520원이 발생합니다.
  • 이벤트 발생 시 사용자에게 통보해주는 SMS는 유료화 예정이며, 유료화할 경우 사전에 충분히 공지하도록 하겠습니다.
  • NAS 상품에 대한 더 자세한 내용은 NAS 상품 소개를 참고하시기 바랍니다.
