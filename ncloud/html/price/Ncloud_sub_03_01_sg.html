Server

운영체제 설치를 위한 기본 디스크가 제공되며, 추가 디스크는 사용자가 원하는 용량 만큼 직접 생성할 수 있습니다.
월 또는 시간 단위로 서버를 이용할 수 있습니다. 월요금의 경우 1개월 미만으로 이용하면 일할 계산되어 청구됩니다.
vCPU 1개, 메모리 2GB 서버는 Linux 계열 운영체제만 사용할 수 있습니다.

서비스 요금 = 월 전송량(transfer) 요금 + 월 요청수(request) 요금
1) 월 전송량(transfer) 요금 = 월 단위 총 CDN 전송량에 대한 사용량 구간별 요금의 합
2) 월 요청수(request) 요금 = 월 단위 총 CDN 요청 수
