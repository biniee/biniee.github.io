요금 계산기

상품 선택 및 사용량 설정을 통해 사용요금을 예상할 수 있습니다

  • 한국
  • 미국
  • 싱가포르
  • 홍콩

Server

다양한 스펙의 서버를 웹 기반의 콘솔로 신속하게 구성할 수 있는 서비스입니다

총 예상 요금(1개월 기준, VAT 별도) 0원 추가하기
  • 운영체제
  • 스토리지 종류
  • 서버 스펙
  • 서버 대수
  • Local Disk 타입의 서버는 서버 요금에 스토리지 요금(운영체제 및 데이터 영역)이 함께 청구됩니다.
  • 서버 스펙 및 상세 요금은 Server 상품소개를 참고하시기 바랍니다.

Block Storage

서버의 데이터를 저장하는 용도로 사용하는 서비스입니다

총 예상 요금(1개월 기준, VAT 별도) 43,200원 추가하기
  • HDD 스토리지 사용량21,600
  • 최소 생성 용량은 10GB이며, 10GB씩 추가할 수 있습니다. 최대 용량은 1000GB입니다.
  • 블록 스토리지 상품에 대한 더 자세한 내용은 Block Storage 상품소개를 참고하시기 바랍니다.

Network

서버를 외부와 연결하거나,
서버로 흐르는 트래픽을 분산하는 목적으로 사용하는 서비스입니다

총 예상 요금(1개월 기준, VAT 별도) 86,400원 추가하기
  • 공인 IP21,600
  • Load Balancer21,600
  • 네트워크(국내 회선) 사용량21,600
  • 네트워크(해외 회선) 사용량21,600
  • 네트워크는 아웃바운드 트래픽만 과금되며, 사용량에 따라 차등을 두어 청구됩니다.
  • 네트워크 상품에 대한 더 자세한 내용은 Network 상품 소개를 참고하시기 바랍니다.

Global CDN

대용량의 콘텐츠를 전세계(한국 제외) 사용자에게 빠르고 손실없이 안전하게 전달하는 서비스입니다

총 예상 요금(1개월 기준, VAT 별도) 11,111원 추가하기
  • 사용량11,111
  • 요청 수11,111
  • Global CDN 요금은 전송량과 요청 수를 합산해서 청구됩니다.
  • 사용량에 따라 차등 청구됩니다.
  • Global CDN 상품에 대한 더 자세한 내용은 Global CDN 상품 소개를 참고하시기 바랍니다.
