특징

네이버의 DBMS인 CUBRID가 설치된 서버 제공

국내 유일의 오픈소스 데이터베이스 관리 시스템인 CUBRID는 네이버와 LINE 등 다양한 서비스에서 운영되며 높은 안정성을 입증했습니다.
대규모 트래픽을 신속히 처리하고, 편리한 사용을 돕는 여러 기능들이 탑재된 CUBRID를 통해 비즈니스의 안정성을 높여보세요.

  • 엔터프라이즈만큼 강력한 DBMS

    CUBRID는 기본적으로 ANSI SQL-92 표준을 따르며, 오라클이 지원하는 계층 질의나 MERGE 문 등 특수한 SQL 구문들을 지원합니다. 응용프로그램 개발을 위해 JDBC, CCI, PHP 등 다양한 API를 제공합니다. 24시간 무정지 및 부하 분산을 위한 고가용성(high availability, HA) 기능을 바로 사용할 수 있습니다.

  • 편리한 사용

    네이버 클라우드 플랫폼에서 지원하는 CUBRID 설치형 서비스는 서버 생성 이후 GUI(Graphic User Interface) 기반의 어드민 툴인 CUBIRD Manager로 데이터베이스 관리, 질의, 모니터링을 할 수 있습니다.

  • 유연한 확장성

    CUBRID는 응용프로그램-브로커-데이터베이스 엔진의 3-tier 구조로 되어 있습니다. 동시 접속자에 따른 대량의 요청을 처리해야 하거나, 대용량 데이터베이스를 운영해야 하는 시점에 수평 확장이 용이하여 보다 안정적인 데이터베이스 인프라를 제공합니다.

  • 실시간 모니터링 가능

    CLI 기반의 모니터링 프로그램을 통해 데이터베이스의 안정적인 운영에 필요한 성능 지표를 실시간으로 제공하고 있으며, 이를 통해 보다 효율적으로 CUBRID를 운영할 수 있습니다.

상세기능

안정화된 버전 제공

네이버 클라우드 플랫폼에서는 네이버와 LINE 서비스에서 검증된 CUBRID 9.2 버전을 제공합니다.

OLTP 환경에서의 고성능 보장

멀티 쓰레드/멀티 서버 구조 설계로 많은 트래픽의 분산처리가 가능합니다. 3tier 구조에서 미들웨어인 브로커를 이용한 커넥션 풀링/로드 밸런싱/Proxy 기능 지원이 가능하며, 데이터베이스 서버에서도 Disk I/O 최적화를 통한 성능 병목 구간 개선 및 고성능 인덱스(Multi-Range/Covered/Reverse/Skip-Scan/Function based/Filtered Index)를 지원합니다.

모니터링 툴(statdump.sh) 제공

네이버 클라우드 플랫폼에서 제공하는 CUBRID 상품은 CLI(Command Line Interface)형태의 모니터링 프로그램이 제공됩니다. 이는 CUBRID DB 상태 변화를 보기 위한 프로그램으로, 1초 단위로 DB Status 를 화면에 출력하여 실시간에 가깝게 DB 상태의 변화를 모니터링 할 수 있어 고객이 안정적이고 빠른 DB 운영을 할 수 있도록 지원합니다.

GUI 환경의 어드민 툴 제공

CUBIRD Manager는 GUI(Graphic User Interface) 도구로 데이터베이스 모니터링 및 관리, 질의 기능이 가능한 CUBRID 데이터베이스 전용 관리도구 입니다.
몇 번의 클릭으로 서버의 상태를 확인 할 수 있으며, 데이터베이스 생성, 구동/정지 및 설정을 변경할 수 있습니다. 응용 개발에 필요한 질의 편집기 도구를 제공해줍니다.
자세한 내용은 아래 URL에서 확인할 수 있습니다.

  • - CUBRID Manager 매뉴얼: http://www.cubrid.org/manual/93/ko/cm/index.html
  • - CUBRID Manager 다운로드: http://ftp.cubrid.org/
  • - _Tools/CUBRID_Manager/

GUI 환경의 마이그레이션 툴 제공

CUBRID Migration Toolkit (CMT)는 GUI 환경 기반에서 스키마 객체 및 데이터를 마이그레이션 하는 도구로서, Oracle 및 MySQL DB를 CUBRID DB로
마이그레이션하는 기능을 제공합니다. 원본 데이터베이스는 Oracle 10g 이상, MySQL 4.1.22 이상 버전을 지원합니다.
해당 기능은 CUBRID Manager의 Migration 메뉴를 통해서도 제공합니다. 자세한 내용은 아래 URL에서 확인할 수 있습니다.

  • - CMT 매뉴얼 : http://www.cubrid.org/wiki_tools/entry/cubrid-migration-toolkit-manual_kr
  • - CMT 다운로드: ftp://ftp.cubrid.org/CUBRID_Tools/CUBRID_Migration_Toolkit
