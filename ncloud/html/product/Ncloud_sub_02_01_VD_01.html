네이버 클라우드 BIZ 본문영역

Virtual Dedicated Server

고성능의 클라우드 서버를 단독 물리 서버에 제공합니다

이용 문의하기
  • 특징
  • 상세 기능
  • 요금
  • 이용안내

특징

단독 물리 서버에 생성된 고성능 클라우드 서버

고성능 물리 서버를 단독으로 제공하는 서비스로,
많은 양의 데이터 처리와 고성능의 I/O를 요구하는 서비스에 적합합니다

  • 고성능 I/O 제공

    SSD가 장착된 물리 서버로 제공되어 안정적인 서버 운영이 필요한 환경에 적합하며, 게임 서버 및 DB 장비로 활용되고 있습니다.

  • 빠른 서버 제공

    일반적으로 고성능의 물리 서버를 제공하기 위해서는 장비 구매에서 서버 준비까지 긴 시간이 필요합니다. Virtual Dedicated Server는 사전에 구축된 물리 서버 풀을 운용하기 때문에 신청 후 수 일 내에 서버를 제공할 수 있습니다. 서버 신청은 고객센터를 통해서 할 수 있으며, 서버가 할당되면 웹 기반 콘솔에서 일반 서버 상품과 동일하게 서버를 생성할 수 있습니다. 단, 다수의 장비가 필요한 경우에는 미리 고객센터에 문의하세요.

  • 효율적인 비용 관리

    사용자는 서버를 사용한 기간 동안만 이용 요금을 지불합니다.
    물리 서버를 사용하기 위한 서버 투자비 및 운영비 없이 고성능 서버를 필요한 기간만큼 사용할 수 있습니다.

이용 문의하기

상세기능

Virtual Dedicated Server는 물리 서버를 클라우드 서버 형태로 제공합니다
일부 기능을 제외하고 일반 서버처럼 서버 기능을 사용하거나 다른 상품에 연동해 사용할 수 있습니다

네이버 클라우드 플랫폼 상품과 연동

Virtual Dedicated Server는 단독으로 제공된 서버 위에서 하나의 클라우드 서버만을 생성해 제공하는 서비스입니다. 물리 서버 제공 방식이지만 클라우드 서비스 형태로 제공되기 때문에 일반 서버처럼 기존의 네이버 클라우드 플랫폼 상품과 연동해 서비스를 이용할 수 있습니다. (단, 내 서버 이미지, 스냅샷, 추가 스토리지, Network Interface 기능 제외)

전용망과의 연결 제공

금융 서비스는 금융 기관과의 전용선 통신 및 기본적인 방화벽을 필요로 합니다. 네이버 클라우드 플랫폼에서는 Virtual Dedicated Server와 금융 기관과의 전용선 통신망을 구축해 금융 서비스가 가능하도록 인프라 구성을 지원하고 있습니다.

이용 문의하기

연동 부가 서비스

  • Monitoring

    보유하고 있는 서버에 대한 종합적이고 세부적인 모니터링 결과를 제공

  • Load Balancer

    서버로 유입되는 네트워크 트래픽을 자동으로 분산
    일부 서버에 장애가 발생해도 중단되지 않음

  • Public IP

    원하는 서버로 할당 및 서버를 변경할 수 있어 서버의 업그레이드와
    다운그레이드, 장애 서버 교체 시에 유용
