SSD Server

높은 I/O 처리가 가능한 SSD가 장착된 서버를 제공합니다

이용 신청하기
  • 특징
  • 상세 기능
  • 요금
  • 이용안내

특징

고성능 I/O를 통한 빠르고 우수한 품질의 서버

SSD Server 는 고성능 I/O를 보장하는 SSD 디스크에 고객이 원하는 운영체제를 설치하여 제공하므로,
빠른 데이터 처리가 필요한 고객에게 적합합니다. 필요시 서버를 수 분 내에 자동으로 생성할 수 있으며,
사용하고 있는 서버의 CPU나 메모리도 스펙 변경 기능을 통해 간단하게 조정할 수 있습니다.

  • 고성능 I/O 제공

    표준 서버보다 10~20배 이상의 고성능 I/O를 제공하여 안정적으로 작업을 처리할 수 있습니다. SSD 서버는 기본 4,000 IOPS(Input/output Operations Per Second)를 보장하며, 스토리지 크기에 비례하여 성능이 증가합니다.

  • 다양한 타입의 스토리지 추가 기능

    서버 생성 시 제공된 기본 저장 공간 외에 추가적인 용량이 필요한 경우, 간편한 용량 확장이 가능합니다. 서버 내 데이터를 보관하기 위해서는 HDD 타입의 저장 공간을, 빠른 데이터 처리를 위해서는 SSD 타입의 저장 공간을 선택하여 구성할 수 있습니다.

이용 신청하기

상세기능

안정적인 성능 제공

별도의 설정 없이 스토리지 할당량에 비례한 IOPS(Input/output operations per second)를 지원합니다. 데이터 처리량이 증가할 때에도 성능 저하 없이 지속성과 안정성을 보장하므로, 높은 I/O 성능이 필요한 작업을 수행할 수 있는 효율적인 플랫폼을 제공합니다.

빠른 응답 처리

SSD(Solid State Drive)는 물리적으로 동작하는 부품이 장착되어 있는 HDD와는 달리 물리적인 동작이 없는, 비휘발성의 플래시 메모리로 구성되어 있는 디스크 미디어입니다. 물리적인 동작이 없는 만큼, HDD에 비해 I/O 처리 시 대기 시간이 매우 짧고 I/O 처리를 위한 대역폭은 굉장히 높습니다. 이에 SSD는 기존 HDD에 비해 I/O에서 발생하는 병목현상을 최소화할 수 있으며, 빠른 응답 속도와 높은 대역폭의 I/O 성능이 동시에 요구되는 서비스에 적합합니다.

IOPS 조절 기능 (출시 예정)

SSD Server는 기본적으로 HDD Server 대비 높은 I/O를 처리할 수 있으며, 갑작스러운 서비스 요청이나 이벤트 등으로 더 많은 I/O 처리가 필요할 경우 손쉽게 IOPS를 상향 조정하여 고객이 필요로 하는 성능을 빠르게 제공할 수 있습니다.

이용 신청하기

연동 부가 서비스

  • Monitoring

    보유하고 있는 서버에 대한 종합적이고 세부적인 모니터링 결과를 제공

  • Load Balancer

    서버로 유입되는 네트워크 트래픽을 자동으로 분산
    일부 서버에 장애가 발생해도 중단되지 않음

  • Public IP

    원하는 서버로 할당 및 서버를 변경할 수 있어 서버의 업그레이드와
    다운그레이드, 장애 서버 교체 시에 유용

  • Storage

    물리 디스크 드라이브와 유사
    저장 용량을 동적으로 추가, 축소 가능

  • Snapshot

    스토리지의 특정 시점 스냅샷을 저장

  • API

    네이버 클라우드 플랫폼에서 제공하는 서비스 및 상품을 이용할 수 있는
    응용프로그램 인터페이스

요금

SSD Server는 표준 서버의 Standard/High Memory 타입과 동일한 스펙의 서버를 제공하며,
서버에서 발생하는 네트워크 트래픽은 표준 서버와 합산하여 과금합니다

SSD Server 요금

Standard/High Memory 타입 상품

  • 운영체제 설치를 위한 기본 디스크가 제공되며, 추가 디스크는 사용자가 원하는 용량만큼 직접 생성할 수 있습니다.
  • 월 또는 시간 단위로 서버를 사용할 수 있습니다. 월 단위 요금으로 서버를 1개월 미만으로 사용하면 이용 요금은 일할 계산되어 청구됩니다.
  • vCPU 1개, 메모리 2GB 서버는 Linux 계열 운영체제만 사용할 수 있습니다.
