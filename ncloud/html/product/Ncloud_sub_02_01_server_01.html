특징

빠르게 생성하여 원하는 분야에 효율적으로 활용하는 서버

신규 사업 시 인프라 보유에 따른 위험 부담을 줄이고자 하는 경우,
일시적으로 많은 서버 자원이 필요한 경우,
인프라 운영에 필요한 인력과 비용 부담을 덜고자 하는 경우에 적합합니다

  • 편리한 서버 생성

    웹 기반 콘솔로 수 분 내에 서버를 간단하게 생성할 수 있으며, 별도의 운영체제를 설치하지 않아도 선택한 서버 이미지에 따라 서버가 제공됩니다. 사용하고 있는 서버의 CPU나 메모리도 스펙 변경 기능을 통해 조정할 수 있습니다.

  • 모니터링 기능 제공

    CPU, 네트워크, 메모리, 디스크에 대한 기본적인 모니터링 기능을 제공합니다.
    Monitoring 상품을 이용하면 서버 상태 변화에 따른 이벤트 알림을 수신할 수 있습니다.

  • 강화된 보안

    네이버 클라우드 플랫폼에서 제공하는 모든 Linux, Windows 서버는 보안 강화를 위해 보안 설정(Security Hardening)이 기본적으로 적용되어 있습니다. 이 중 Windows는 OS 특성을 고려하여 바이러스, 악성코드 등을 탐지/방어하기 위한 목적으로 서버 생성 시 백신이 무료로 설치됩니다. 기본 보안 설정 이외에 엄격한 보안 설정 적용에 대한 컨설팅 및 문의가 필요할 경우 보안 전문가의 도움을 받을 수 있습니다.

  • 효율적인 비용 관리

    서버를 사용한 용량, 시간만큼만 비용을 지불하기 때문에 계획적인 비용 관리가 가능합니다. 서비스 운영 중에도 편하게 서버를 추가 투입하거나 반납할 수 있어, 비용을 효율적으로 관리할 수 있습니다. 다양한 상황에도 빠르고 탄력적으로 대응할 수 있습니다.

이용 신청하기

상세기능

서버를 쉽고 편리하게, 안정적으로 사용할 수 있는 다양한 기능을 제공합니다

Init-Script 기능

사용자가 원하는 서버 설정 및 수행 작업을 스크립트로 작성하여, 서버가 생성될 때 자동으로 실행하는 Init-Script 기능을 제공합니다. 스크립트는OS에 맞는 언어로 작성하여 웹 기반의 콘솔에서 등록하거나 서버 RESTful AP를 통해 적용할 수 있습니다.

Init-Script
운영체제 스크립트 언어
Linux 계열 Python, Peri, 셸 스크립트
Windows 계열 Visual Basic 스크립트

서버 간 통신

사용자가 보유하고 있는 서버 사이에 사설 통신이 가능합니다. 또한 외부에서 서버에 접속하기 위한 접속용 공인 IP를 제공합니다. 별도의 공인 IP를 신청해 서버에 할당하면 외부에 서비스를 제공할 수 있습니다. 외부 서비스를 위해서는 직접 공인 IP를 할당하는 방법 외에도 Load Balancer 상품에 서버를 연결해 사용하는 방법도 있습니다.

추가 네트워크 인터페이스 지원

서버에 추가 Network Interface를 생성해 서버 사이의 내부 통신을 위한 독립된 가상 네트워크를 구성할 수 있으며, HA(high availability) 구성 시에 서버 사이의 전용 네트워크 연결로 활용할 수 있습니다. CIDR 형식으로 192.168.0.0/16 미만의 IP 주소 대역 중 /24~/26까지의 서브넷 비트만 설정할 수 있습니다. 네트워크 주소를 포함해 초기 11개의 IP 주소는 관리용으로 예약됩니다.

공인 IP 제공

사용자가 보유하고 있는 어떤 서버에도 연결될 수 있는 고정된 IP 주소인 공인 IP를 제공합니다. 공인 IP는 사용자가 지정한 서버에 할당할 수 있습니다. 할당된 공인 IP는 필요에 따라 사용자가 보유한 다른 서버로 변경해 할당할 수 있습니다. 기존 서버를 신규 서버로 이전할 때, 준비된 신규 서버에 기존과 동일한 환경을 구축한 후 기존 서버의 공인 IP를 신규 서버에 할당하기만 하면 짧은 서비스 중단 시간 이후 서비스를 연속적으로 제공할 수 있습니다.

내 서버 이미지 기능의 제공

현재 운영되고 있는 서버와 동일한 데이터를 가지는 서버를 만들 때, 현재의 서버 상태를 이미지로 저장한 후 저장된 이미지를 복제해 서버를 생성할 수 있습니다. 서버 이미지를 생성한 시점의 상태로 서버가 복구되며, 서버 스펙도 변경할 수 있습니다.

다양한 서버 이미지

다양한 버전의 운영체제를 선택할 수 있습니다. DBMS 상품으로는 MySQL이나 MSSQL이 설치된 서버가 제공됩니다.
(단, DBMS는 해당 소프트웨어 제공사의 라이선스 정책에 따라 과금합니다)

  • MSSQL(Microsoft SQL Server)을 사용할 경우에는 다음 내용을 참고하시기 바랍니다.
    • - 라이선스 1 copy로 vCPU 4개를 지원합니다. vCPU가 1~4개인 서버는 1 copy를, vCPU가 8개인 서버는 2 copy를, vCPU가 12개인 서버는 3 copy를 구매해야 합니다.
    • - MSSQL 이용 요금은 DBMS 라이선스 요금이며, 서버와 운영체제(Windows)의 이용 요금은 별도로 부과됩니다.
    • (예: 2 vCPU 서버에서 MSSQL이용시, 69,000원(서버) + 20,000원(Windows 라이선스) + 290,000원(MSSQL 라이선스) =379,000원/월)
    • - Microsoft 사의 정책에 따라 이용 요금이 변경될 수 있으며, 기존에 사용하고 계신 사용자에게도 새로운 요금이 적용됩니다.
