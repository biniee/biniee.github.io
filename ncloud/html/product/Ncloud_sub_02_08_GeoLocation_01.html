GeoLocation

IP 주소에 대한 지역 정보를 검색해
사용자 위치에 따른 맞춤형 콘텐츠를 제공합니다

이용 신청하기

  • 상세 기능
  • 요금
  • 이용안내

특징

사용자 IP 기반 위치 정보를 제공하는 국내 유일의 서비스

GeoLocation은 고객 서버에서 질의한 IP 주소에 따른 지역정보 DB(GeoLocation DB)를 검색해
국가, 시/군/구, 동, 인근 지역의 좌표, 이용 통신사 등의 정보를 전달합니다. 이 상품을 이용해 사용자 위치에 따른
차별화된 콘텐츠 제공, 광고 타겟팅, 트래픽 분석, DRM 관리, 어뷰징 및 Fraud 탐지 등을 할 수 있습니다.

  • 검증된 상품성

    네이버 검색 광고, 날씨, 지도, 동영상 등 다양한 서비스에서 이미 GeoLocation을 사용하고 있습니다. 검색자의 현재 위치와 관련된 광고를 노출하거나, 사용자의 현재 위치에 대한 날씨를 관심 지역으로 제안합니다. 이처럼 네이버에서 오랜 기간 사용하며 검증한 서비스를 직접 활용해 볼 수 있습니다.

  • 정확도 높은 최신 위치 정보 제공

    GeoLocation를 통해 국가는 물론 시/군/구와 동, 인근 지역의 좌표 정보를 얻을 수 있습니다. 이러한 정보는 네이버에서 수집되는 각종 데이터를 바탕으로 만들어집니다. IP별 위치 변경이 감지될 때마다 매일 수시로 갱신되기 때문에 높은 정확도를 유지하는 GeoLocation DB로부터 늘 최신의 정보를 제공받을 수 있습니다.

  • 맞춤형 서비스 가능

    사용자의 IP 주소 위치에 따라 차별화된 콘텐츠를 제공하기 때문에 더욱 효과적인 서비스가 가능합니다. 고객은 지역별로 제공할 콘텐츠를 미리 DB화 하여, GeoLocation API의 결과를 이용해 적절한 정보를 사용자에게 맞춰서 제공할 수 있습니다.

  • 편리한 사용

    RESTful API를 이용하여 GET 혹은 POST 방법으로 XML, JSON 형태의 원하는 정보를 쉽게 얻을 수 있으므로, 간편하게 고객의 서비스에 적용할 수 있습니다. 앞으로 다양한 GeoLocation API를 추가로 제공할 예정이므로 편의성은 더욱 확대될 것입니다.

제공 항목

사용자 위치에 관한 상세한 정보를 제공하므로 다양한 활용이 가능합니다. 이 외에도 지역과 관련된 다양한 정보를 API로 추가 제공할 예정입니다.
(추가가 필요한 정보가 있다면 고객센터를 통해 제안해주시면 검토하겠습니다.)

API 사용

API를 사용하려면 OAuth 인증 방식의 인증키를 발급 받아야 합니다. API 인증키는 네이버 클라우드 플랫폼 포털 사이트의 ‘마이페이지-인증키 관리’에서 발급받고 관리할 수 있습니다. API는 RESTful API 방식의 HTTP GET/POST 메서드 호출로 실행하며, API를 사용해 원하는 형태의 모니터링 관리 기능을 만들 수 있습니다.

사용 상한 설정

사용 상한(Quota)을 정하여 의도치 않게 과도한 사용을 하지 않도록 미리 설정할 수 있습니다. 과금은 월 API 호출 수에 따르므로, 월 API 호출 수에 대해 사용 상한을 설정할 수 있습니다. 만약 실제 API 호출 수가 사용 상한에 다다를 경우, SMS/Mail 알림을 받도록 설정할 수 있습니다. 필요에 따라 언제든지 사용 상한을 변경할 수 있어 효율적인 비용 관리가 가능합니다.

기타 지표

성능 및 안정성

NPC망 내에서 10msec 이내의 응답시간을 제공하며, 최근 1년간 네이버에 서비스를 제공하는 동안 1sec의 downtime도 없이 운영하는 정도의 안정성을 가지고 있습니다. 그리고, 요청량에 따라 점진적으로 용량을 늘려갈 계획이므로, 가용한 용량에 상한선이 없다고 느끼실 것입니다.

글로벌 리전

현재는 한국에만 구축되어 있으나 글로벌 리전에서도 사용은 가능합니다. 다만 글로벌 리전에서 사용하면 응답시간이 증가할 수 있습니다. 만약 글로벌 리전에서 GeoLocation을 사용하고자 하는 경우에는 고객센터를 통해 요청주시면 글로벌 리전에도 구축할 계획을 갖고 있습니다.
