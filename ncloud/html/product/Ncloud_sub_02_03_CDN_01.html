특징

다양한 콘텐츠를 빠르고 안전하게 전달

CDN(Content Delivery Network)은 인터넷 사용자에게 다양한 콘텐츠를 빠르고 안전하게 전달하는 서비스입니다.
수많은 이미지 콘텐츠의 빠른 전송, 대용량 콘텐츠의 초고속 다운로드, 미디어 콘텐츠의 끊김 없는 스트리밍 서비스 등 많은 분야에서 활용됩니다.
웹 기반의 콘솔로 편리하게 CDN 서비스를 이용할 수 있습니다. CDN 서비스를 신청하면 수 분 내에 서비스 제공이 완료돼 원하는 콘텐츠를 전달할 수 있습니다.
콘텐츠를 전송하기 위해 사용한 네트워크 및 요청수만큼 요금이 부과되는 상품이라 사용한 만큼만 비용을 지불합니다.

  • 웹캐싱, 다운로드, 스트리밍 서비스 동시 제공

    이미지, 게임 설치 파일 등 전통적인 웹캐싱, 다운로드 서비스뿐만 아니라, 대용량 미디어 콘텐츠에 대해 HLS(HTTP Live Streaming), MPEG-DASH 등과 같은 HTTP/HTTPS 표준 규격 기반의 스트리밍 전송 기능을 동시에 제공합니다.

  • 간단한 신청 및 활용

    고객이 온라인으로 CDN 서비스 생성, 설정할 수 있도록 웹 기반의 콘솔을 제공합니다. CDN 서비스 생성 후, 수 분 이내에 CDN 제공이 완료되어, 바로 콘텐츠 전송이 가능합니다. 고객의 CDN 이용금액은 콘텐츠 전송량 및 요청 수에 비례하며, 서비스 이용에 따른 사용량 약정이 없습니다.

  • 네이버 클라우드 플랫폼 타 상품과 연계 가능

    타 상품과의 편리한 연동 설정 기능을 제공합니다. File Storage, Server 상품 사용 고객은 CDN 원본 서버 및 CDN 로그 저장소로 기존에 사용 중인 상품을 활용할 수 있습니다.

  • 신속한 이벤트 알림

    고객의 CDN 사용 패턴 확인과 사용량 관리를 위해, CDN 사용량에 대한 알림 기능을 고객에게 제공 합니다. 전송량, 호출수에 대해 월 누적 사용량이 고객이 설정한 값을 초과하면 메일 또는 SMS로 안내됩니다.

이용 신청하기

상세기능

Server, File Storage 기능과 연계하여 쉽고 편리하게 사용할 수 있습니다

콘텐츠 보안 기능

다양한 콘텐츠 보안 기능을 제공하여, 안전하게 콘텐츠를 전송합니다. 콘텐츠 보안을 위해 제공하는 기능은 다음과 같습니다.

  • HTTPS 전송 지원 (SSL 인증서 설정 기능 포함)
  • 등록된 도메인 기준 접근 허용 기능
  • Secure-Token 기반 보안 URL 생성 기능

웹캐싱, 다운로드 기능

웹캐싱, 다운로드와 같은 일반적인 CDN 서비스 속성을 웹 기반의 콘솔을 통해, 간편하게 설정할 수 있습니다. HTTP/HTTPS 프로토콜로 서비스를 제공하며, Cache Expire, Ignore Query String, Referrer Domain 등의 캐싱 옵션을 지원합니다.

스트리밍

대용량 미디어 콘텐츠에 대한 HTTP 표준 규격 기반의 스트리밍 방식의 콘텐츠 전송을 제공합니다. 단, 미디어 콘텐츠는 MP4/MOV/MP3 (Video H.264/AVC, Audio AAC/MP3) 규격을 따라야 합니다. 이러한 미디어 콘텐츠에 대해 HLS (HTTP Live Streaming), MPEG-DASH, Pseudo-Streaming 방식으로 사용자에게 전송합니다.

CDN 통계/모니터링 기능

다음과 같은 항목들을 기본으로 제공합니다.

  • 이용량 (전송량/요청수) 통계
  • 응답 코드별 통계
  • 국가별 접속 통계
  • Referer 통계

콘텐츠 접근 로그 제공

고객의 콘텐츠 전송 및 사용 패턴 분석 지원을 위하여 CDN 콘텐츠 요청에 대한 접근 로그를 제공합니다. 추출된 접근 로그는 File Storage 와 연동되어 주기적으로 자동 저장됩니다.

다양한 원본 서버 지원

CDN은 HTTP 프로토콜을 통해 다양한 원본 서버와의 콘텐츠 연동을 지원 합니다. 원본 서버로 고객이 보유중인 다양한 웹 서버 및 네이버 클라우드 플랫폼의 File Storage, Server를 설정할 수 있으며, 연결을 위한 포트를 자유롭게 지정할 수 있습니다.
